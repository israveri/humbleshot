# Humbleshot

Simple tool for screenshots and annotations.

---

### Dependencies

* [grim](https://gitlab.freedesktop.org/emersion/grim) for grabbing images from the compositor and saving to file

* [slurp](https://github.com/emersion/slurp) for selecting a region of the screen to grab

* [satty](https://github.com/Satty-org/Satty) for annotations

---

### Integrations

This tool integrates with Noctalia's toast notifications with fallback to `notify-send`
